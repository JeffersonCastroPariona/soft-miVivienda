# Curso: Finanzas e Ingeniería Económica

## Descripción del Proyecto
Proyecto académico desarrollado como parte del curso **Finanzas e Ingeniería Económica**.

El proyecto consiste en la **implementación de un sistema de cotizaciones**, permitiendo calcular y gestionar costos mediante el uso de **distintos métodos de pago**, con el objetivo de apoyar la toma de decisiones financieras.

## Tecnologías Utilizadas
- **TypeScript**
- **React** (Frontend)
- **Node.js** (Backend)

## Funcionalidades Principales
- Generación de cotizaciones dinámicas  
- Gestión de múltiples métodos de pago  
- Cálculo automático de montos y totales  
- Interfaz interactiva y responsive  

## Objetivo Académico
Aplicar conceptos de:
- Análisis financiero
- Ingeniería económica
- Evaluación de alternativas de pago
- Desarrollo de software web moderno

## Rol
- Desarrollo frontend y backend
- Implementación de la lógica de cotización
- Integración de métodos de pago
